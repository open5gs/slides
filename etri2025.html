What is Open5GS?

  • Open Source 4G/5G Core Network
    • Complete EPC (4G) and 5G Core implementation
  • Started as Personal Project in 2017
    • Originally NextEPC, forked to Open5GS in 2019
  • 8+ Years of Continuous Development
    • Proven stability and maturity through long-term evolution
  • Independent Community Project
    • No external funding or corporate backing since inception

Key Technical Achievements

Standards & Core Archi

  • Microservices: Cloud-native design
  • Interoperability: Works with commercial RAN

Security & Operations

  • SUCI Protection: Profile A/B encryption

  • Monitoring: Prometheus integration

🎉 Recent Major Achievement: Complete 5G Roaming

  • Full 5G roaming functionality merged to main branch!
    • Both LBO (Local Breakout) and HR (Home Routed) supported
  • Built-in test framework included
    • Self-contained test environment since UERANSIM lacks roaming support
  • Comprehensive tutorial guide
    • Step-by-step guide from build to configuration and testing
  • Packet capture examples provided
    • Real traffic analysis materials for LBO/HR scenarios

5G Roaming Architecture Implementation

  • SEPP Fully Implemented
    • Security Edge Protection Proxy
  • N32 Interface
    • Secure TLS communication between operators
  • FQDN-based Routing
    • Automatic routing via PLMN identifiers
